# 📖 Purpose of this Lab

Modern applications face increasingly sophisticated cyber threats that require proactive security planning before software reaches production. Threat modeling is one of the most effective methods for identifying security weaknesses during the design phase, allowing organizations to reduce risk early in the Software Development Life Cycle (SDLC).

This lab introduces students to industry-standard threat modeling methodologies by combining **Draw.io**, **MITRE ATT&CK Navigator**, and **Python automation**. Students will learn how to visualize application architectures, identify trust boundaries, model attack paths, and automate the generation of security documentation using real-world threat intelligence.

Rather than relying solely on manual documentation, this lab demonstrates how automation can streamline threat modeling processes while integrating the MITRE ATT&CK framework to create repeatable and scalable security assessments.

By completing this exercise, learners will gain practical experience in designing secure architectures, documenting attack scenarios, prioritizing security risks, and producing professional threat models suitable for product security reviews, secure design assessments, and supply chain assurance activities.
