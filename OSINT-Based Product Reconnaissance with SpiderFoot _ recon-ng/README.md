# 🕵️ OSINT-Based Product Reconnaissance with SpiderFoot & recon-ng

> **Learn how to perform automated Open Source Intelligence (OSINT) reconnaissance to discover publicly exposed information about products, domains, and organizational assets using SpiderFoot, recon-ng, Python automation, and DNS analysis.**

---

## 🎯 Purpose of this Lab

This lab is designed to introduce learners to **Open Source Intelligence (OSINT)** techniques for product security and external attack surface discovery. Students will learn how to collect publicly available information about target organizations, products, domains, and internet-facing assets using industry-standard reconnaissance tools.

Throughout the lab, participants will configure **SpiderFoot** and **recon-ng**, automate reconnaissance workflows using **Python**, perform DNS enumeration, analyze collected intelligence, and generate professional security reports. The exercises demonstrate how security professionals identify publicly exposed assets, discover potential security risks, and improve an organization's external security posture.

The lab emphasizes practical automation, allowing students to build reusable reconnaissance scripts while understanding how OSINT supports penetration testing, red teaming, threat intelligence, vulnerability assessments, and product security reviews.

# 🌟 Why This Lab Matters

Modern organizations expose a significant amount of information through public infrastructure, cloud services, source code repositories, and internet-facing applications. Security professionals rely on OSINT techniques to identify these assets before attackers do.

This lab teaches how to automate reconnaissance, analyze publicly available intelligence, and transform raw data into actionable security insights. These skills are fundamental for product security assessments, threat intelligence operations, penetration testing, vulnerability management, and continuous attack surface monitoring.
