# 📊 Process Product Telemetry Data in Jupyter + Pandas

> **Hands-on Product Security Lab**  
> Learn how to collect, process, analyze, and visualize product telemetry data using **Jupyter Notebook**, **Python**, **Pandas**, and **Matplotlib** while building automated analytics workflows for continuous product monitoring.

---

## 📖 Overview

Modern software products, IoT devices, and cloud services continuously generate telemetry data that provides valuable insight into application health, performance, reliability, and potential security issues.

In this lab, students learn how to transform raw telemetry data into actionable insights using the Python data science ecosystem. Beginning with synthetic telemetry generation, learners clean and process datasets, perform statistical analysis, detect anomalies, build visualizations, and automate the entire workflow.

The lab emphasizes practical skills used by Product Security Engineers, DevSecOps Engineers, Data Analysts, Reliability Engineers, and Security Operations teams.
