# 🌐 Trace Product Network Activity Using ProcDOT & Wireshark

> **Purpose Repository** for the **Product Security & Supply Chain Assurance** learning path.

---

# 🎯 Purpose

This repository provides a hands-on laboratory for learning how to capture, analyze, visualize, and automate network traffic analysis using **Wireshark**, **TShark**, **ProcDOT concepts**, and **Python**.

The lab enables students to understand how software products communicate across networks, identify suspicious network behaviors, correlate processes with network connections, and build automated workflows for network traffic monitoring and security analysis.

Students will gain practical experience with packet capture, protocol analysis, traffic visualization, process correlation, and automation techniques commonly used by Product Security Engineers, Security Analysts, and Incident Response teams.

---

# 📚 What You Will Learn

Throughout this lab you will learn how to:

- 🌐 Capture network traffic using Wireshark and TShark
- 📡 Analyze TCP/IP communications and protocols
- 🔍 Monitor application network behavior
- 🧩 Correlate process activity with network connections
- 🐍 Develop Python scripts to automate traffic analysis
- 📊 Visualize network communication flows
- 🚨 Detect suspicious and anomalous network activities
- 📄 Generate structured security reports
- ⚙️ Build an end-to-end automated network analysis pipeline
